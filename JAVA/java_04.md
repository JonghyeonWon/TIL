# JAVA

## 변수와 메서드
클래스를 구성하는 핵심 두 가지 요소

__클래스의 구성요소__
- __클래스 = 변수(데이터) + 메서드(행위)__

__변수의 구문 형식 및 선언 위치__
- __구문 형식__: `[접근지정자](생략 가능) [변수타입] [변수명] = value;` 
- __전역변수__: 클래스 블록 내부에 선언되며, 클래스 내 어디서든 사용 가능
- __지역변수__: 메서드 블록 내부에 선언되며, 지역 변수를 포함하는 블록 안에서만 사용 가능

__메서드__
- 데이터를 다루는 행위
- **구문 형식:**
```text
  [접근지정자][반환타입][메서드명](매개변수1, 매개변수2, ...) {

  }
```

- 메서드의 유형: 반환타입과 매개변수 유무에 따른 4가지 유형 존재

1) 매개변수 존재 + 반환 값 존재
2) 매개변수 존재 + 반환 값 없음
3) 매개변수 없음 + 반환 값 존재
4) 매개변수 없음 + 반환 값 없음

상황에 따라 유동적으로 4가지 유형에서 골라서 사용

__변수 및 메서드 통합 예제__

```java
public class VariableApp {

    // 전역변수가 위치 
    public String  name       = "jhwon";    // 문자열 ("")
    public int     age        = 24;         // 정수
    public double  height     = 165.5;      // 실수
    public boolean isMarriage = false;      // 불린 타입
    public char    gender     = 'm';        // 문자 ('')
    
    // 메서드 예시
    public static void main(String[] args) {
        
        // 지역변수가 위치 (main 메서드 블록 안에서만 사용 가능)
        String msg = "이 곳은 프로그램의 시작점입니다.";
    }
}
```
